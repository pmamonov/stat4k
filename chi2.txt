Первая задача - продемонстрировать, что значение статистики хи-квадрат
имеет распределение хи-квадрат и выходит за рамки "критического" в одном
случае из 20.
1. Необходимо сгенерить выборки трех размеров: 10, 100 и 1000 элементов, по
1000 шт. каждого размера. Закон распределения случайной величины:
вероятность получить 0 равна 0,75, вероятность получить 1 равна 0,25 (типа
это расщепление 3:1). В итоге необходимо получить число нулей и единиц в
каждой выборке.
2. Вычислить для каждой выборки два значения статистики хи-квадрат, одно
для заведомо верной H0 (нули:единицы 3:1), второе - для неверной H0
(нули:единицы 13:3). Хи-квадрат рассчитывается так:

хи-квадрат = (набл.нули - ож. нули)^2/ож.нули + (набл.ед - ож. ед)^2/ож.ед

набл - значение из выборки, ож - значение, рассчитанное из закона
распределения. Например для выборки 73 нуля, 27 единиц и гипотезы 3:1
хи-квадрат равен:

хи-квадрат = (73 - 75)^2/50 + (27 - 25)^2/25

3. Построить распределение вероятностей получения разного количества нулей
(по X-количество нулей, по Y-доля выборок с этим количеством нулей).
Построить распределение значений хи-квадрат для верной и неверной гипотезы
для групп выборок трех разных размеров. Шаг по оси X для верной гипотезы -
0,2; для неверной - какой придется. Также надо представить данные, сколько
выборок каждого размера имеют значение хи-квадрат больше 3,841 (для обеих
гипотез) и максимальное полученное значение хи-квадрат для каждого размера
выборки.

Предполагается увидеть, что:

С увеличением размера выборки вероятность получить ровно 75/25 падает.
Значение статистики хи-квадрат имеет распределение хи-квадрат и практически
не зависит от размера выборки, если он достаточно большой.
Значения более критического (3,841) получаются с вероятностью 0,05.
На выборках малого размера это происходит реже и форма распределения другая.
Способность отвергнуть неверную гипотезу сильно зависит от размера выборки.
